# 🚀 AI Credit Underwriting System

An **AI-powered financial analysis platform** that converts raw
financial documents into a **credit risk assessment report**.

This system automates the traditional credit underwriting process by
extracting financial information from uploaded documents, predicting
**credit risk using machine learning**, and generating an **AI-backed
recommendation** using a Large Language Model (LLM).

# 🛠 Installation (Local Setup)

Clone the repository:

``` bash
git clone https://github.com/yourusername/creditAI.git
```

Move into the project folder:

``` bash
cd credit-underwriting-ai
```

Install dependencies:

``` bash
pip install -r requirements.txt
```

Run the application:

``` bash
python app.py
```

Open in browser:

    http://localhost:5000

# ☁️ Deployment

This project is deployed using **Hugging Face Spaces**.

Steps:

1.  Create a Hugging Face Space\
2.  Select **Docker SDK**\
3.  Push project files to the Space repository\
4.  Hugging Face automatically builds and deploys the application

Example live URL:

https://huggingface.co/spaces/ImShekhuu/CreditAI
